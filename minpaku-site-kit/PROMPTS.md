# 🤖 コピペ用プロンプト集

AI（**ChatGPT / Claude / Claude Code** など）に、このサイトを作って・直してもらうための
「指示文（プロンプト）」をまとめました。

## 使い方
1. 下のグレーの枠（コード）を**まるごとコピー**する
2. AIのチャット欄に**貼り付ける**
3. `【　】` の部分を**自分の物件の情報に書き換える**
4. 送信する

> 💡 迷ったら **プロンプト①** だけでOK。サイトが丸ごと1つ出てきます。

---

## ① サイトを丸ごと作ってもらう（いちばん最初にこれ）

```
あなたはプロのWebデザイナーです。
民泊・宿の「公式サイト」を、HTMLファイル1つ（CSSとJavaScriptを内蔵）で作ってください。
ReactなどのフレームワークやビルドツールはNG。.htmlをダブルクリックすれば動く形でお願いします。

【必須の作り】
- 1ファイル完結（外部はGoogle Fontsの読み込みのみOK）
- スマホ・PC両対応（レスポンシブ）。スマホはハンバーガーメニュー＋画面下に固定の予約ボタン
- 多言語切替（右上のセレクトボックス）。テキストは data-i18n 属性 ＋ JSオブジェクトで管理し、選んだ言語は localStorage に保存
- スクロールで各セクションがふわっと表示されるアニメーション
- 写真がまだ無い場合は、SVGのプレースホルダ（グレーの枠＋アイコン＋ラベル）を自動表示する関数を入れる

【ページ構成（この順番）】
1. ヘッダー（ロゴ＝物件名／ナビ／言語切替）
2. ヒーロー（全画面の写真が数秒ごとに自動で切り替わる＋物件名＋キャッチコピー＋予約ボタン）
3. 物件紹介（短い紹介文＋特徴4つのカード：定員・寝室数・特徴2つ）
4. 写真ギャラリー（カテゴリで絞り込みボタン＋写真クリックで拡大表示するライトボックス）
5. 設備・アメニティ（アイコン付き8項目）
6. アクセス（左：Googleマップ埋め込み／右：最寄り駅などの行き方＋周辺スポット4つ）
7. 予約（Airbnb・Booking.comへのボタン＋問い合わせメール）
8. フッター（物件名・住所・SNS・コピーライト）

【対応言語】
日本語・英語【＋必要なら 中国語・韓国語 などを追加】

【私の物件情報】
- 物件名：【例：海辺の宿 ○○】
- 場所：【例：和歌山県白浜町】
- 最寄り：【例：JR白浜駅から車で10分】
- 定員・寝室：【例：最大8名・寝室3室】
- 雰囲気・ウリ：【例：オーシャンビュー、源泉かけ流し、ペット可】
- 設備（8つほど）：【例：Wi-Fi、フルキッチン、BBQ、駐車場…】
- 周辺スポット（4つ）：【例：白良浜、アドベンチャーワールド…】
- 配色のイメージ：【例：海をイメージした青と白／和モダンの茶色 など】
- 予約リンク：【AirbnbのURL】【Booking.comのURL】

まず完成したHTMLを丸ごと出力してください。そのあと、写真の入れ方を3行で教えてください。
```

## ② 配布テンプレート（template.html）を埋めてもらう

> すでに `template.html` を持っている場合に、中身だけ埋めてもらう使い方です。

```
添付（または下に貼り付け）した template.html は、民泊サイトの白紙テンプレートです。
このファイルの構造・デザイン・機能は変えずに、i18nData の中の【　】部分だけを
私の物件情報で埋めて、完成版のHTMLを丸ごと出力してください。

【私の物件情報】
- 物件名：【　】
- キャッチコピー：【　】
- 紹介文（3〜4行）：【　】
- 特徴4つ：【定員】【寝室数】【特徴1】【特徴2】
- 設備8つ：【　】
- アクセス3つ：【　】
- 周辺スポット4つ：【　】
- 予約リンク：【　】

日本語(ja)と英語(en)の両方を、自然な表現で埋めてください。英語は直訳ではなく、宿泊予約サイトらしい魅力的な英語でお願いします。
```

## ③ 写真の入れ方を手伝ってもらう

```
このHTMLの写真を、仮画像から自分の写真に差し替えたいです。
私の写真ファイルは images フォルダに入れます。ファイル名は次のとおりです：
【例：living.jpg, bedroom.jpg, kitchen.jpg, exterior.jpg ...】

ヒーロー（最初の大きな写真）とギャラリーの <img> を、私のファイル名に差し替えた
HTMLを出力してください。どの写真がどこに入るかも一覧で教えてください。
```

---

## ④ 多言語（翻訳）を追加してもらう

```
このHTMLは今、日本語と英語に対応しています。
ここに【中国語（簡体字）／韓国語 など】を追加してください。

- 言語切替のセレクトボックスに選択肢を追加
- i18nData に同じキー構成で新しい言語を追加
- 翻訳は機械的な直訳ではなく、その言語の宿泊予約サイトとして自然で魅力的な表現に

修正後のHTMLを丸ごと出力してください。
```

---

## ⑤ 配色（カラー）を変えてもらう

```
このHTMLの配色を変えたいです。物件の雰囲気は【例：海沿いのリゾート／古民家の和モダン】です。
似合うカラーパレットを2案提案し、それぞれ :root の色コード（--primary, --accent, --bg など）を
コピペできる形で出してください。おすすめ1案で全体を修正したHTMLも出力してください。
```

---

## ⑥ キャッチコピー・紹介文を考えてもらう

```
民泊「【物件名】」のサイトに載せる文章を考えてください。
物件の特徴：【例：駅徒歩1分、夜景、デザイナーズ、最大6名】
ターゲット：【例：海外からの観光客、記念日カップル】

- ヒーローのキャッチコピー（短め）を3案
- 物件紹介文（3〜4行）を1案
日本語と英語の両方でお願いします。
```

---

## ⑦ 周辺の観光情報を埋めてもらう

```
【物件の住所：例 神奈川県逗子市】の近くにある、観光客に人気のスポットや
おすすめの飲食店を、徒歩・車での所要時間つきで8つ挙げてください。
そのうち4つを、サイトの「周辺スポット」欄に載せる短い表現にまとめてください。
```

## ⑧ うまく動かないとき（エラー解決）

```
このHTMLの【例：言語切替ボタン／写真の拡大表示】がうまく動きません。
HTMLを全部貼ります。原因と、直し方（修正後のコード）を、初心者にもわかるように教えてください。
```
