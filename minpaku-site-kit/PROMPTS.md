# 🤖 コピペ用プロンプト集（ハウスガイド）

AI（**ChatGPT / Claude / Claude Code** など）に、ゲスト向けハウスガイドを
作って・直してもらうための「指示文（プロンプト）」をまとめました。

## 使い方
1. 下のグレーの枠（コード）を**まるごとコピー**する
2. AIのチャット欄に**貼り付ける**
3. `【　】` の部分を**自分の物件の情報に書き換える**
4. 送信する

> 💡 迷ったら **プロンプト①** だけでOK。ハウスガイドが丸ごと1つ出てきます。

---

## ① ハウスガイドを丸ごと作ってもらう（いちばん最初にこれ）

```
あなたはプロのWebデザイナーです。
民泊・宿の「ハウスガイド（滞在中のゲストがスマホで読む電子マニュアル）」を、
HTMLファイル1つ（CSSとJavaScriptを内蔵）で作ってください。
ReactなどのフレームワークやビルドツールはNG。.htmlをダブルクリックすれば動く形でお願いします。

【必須の作り】
- 1ファイル完結（外部はGoogle Fontsの読み込みのみOK）
- スマホで読むことが大前提（レスポンシブ）。スマホでは目次をハンバーガーで開閉
- 5言語切替（日本語・英語・中国語・韓国語・フランス語）。各テキストは
  <span class="l-ja">…</span><span class="l-en">…</span> のように言語別spanで持ち、
  body[data-lang="xx"] のCSSで表示を出し分け。選んだ言語は localStorage に保存
- 画面上部に固定の言語切替バー、左側に「目次サイドバー」（各セクションへジャンプ）
- 情報を種類で色分けする「ブロック」を用意：緊急=赤／注意=橙／情報=青／OK=緑
- 鍵の暗証番号やWi-Fiは、目立つ「バッジ」で表示
- 写真がまだ無い場所は、グレーのプレースホルダ枠（アイコン＋ラベル）を表示

【セクション構成（この順番・不要なものは省いてOK）】
1. 表紙（物件名＋ひとことキャッチ）
2. ようこそ（短い挨拶・基本情報）
3. ハウスルール（禁止事項・お願い）
4. 連絡先（ホスト・緊急時の連絡方法）
5. アクセス（住所・最寄り駅からの行き方・地図リンク）
6. チェックイン（入り方・鍵/暗証番号の使い方）
7. 設備の使い方（エアコン・お風呂・キッチン家電など）
8. Wi-Fi（SSID・つなぎ方）
9. ゴミの分別・出し方
10. 周辺情報（コンビニ・スーパー・飲食店）
11. チェックアウト（手順・退室時のお願い）
12. 緊急時（地震・火事・体調不良・最寄りの病院・避難場所）

【私の物件情報】
- 物件名：【例：湖畔の宿 ○○】
- 場所・最寄り：【例：長野県○○高原 / ○○駅からタクシー10分】
- 定員・寝室：【例：最大8名・寝室3室】
- チェックイン/アウト：【例：15:00 / 10:00、暗証番号は予約メッセージで通知】
- Wi-Fi：【SSIDとパスワード ※載せ方は後述】
- ハウスルール：【例：禁煙、22時以降は静かに、ペット応相談…】
- ゴミ：【例：燃える/缶ビン/ペット、収集日…】
- 周辺：【例：コンビニ、スーパー、おすすめ飲食店…】
- 緊急連絡先：【ホスト電話、最寄り病院、避難場所 ※実在の正確な情報】
- 配色のイメージ：【例：森と湖の緑＋木の温かみ／海の青と白 など】

まず完成したHTMLを丸ごと出力してください。そのあと、写真の入れ方を3行で教えてください。
```

【私の物件情報】

- チェックイン/アウト：【　】
- 鍵・暗証番号の扱い：【　】

## ③ セクションを追加・削除してもらう

```
このハウスガイドHTMLに、新しいセクション「【例：BBQ／温泉／ペットのお願い】」を追加してください。
- サイドバー目次にも項目を追加
- 既存セクションと同じHTML構造・class（色分けブロック等）を使う
- 5言語すべてに対応
逆に「【例：駐車場】」のセクションは不要なので、目次ごと削除してください。
修正後のHTMLを丸ごと出力してください。
```

## ④ 多言語（翻訳）を追加してもらう

```
このハウスガイドHTMLは今、日本語・英語・中国語・韓国語・フランス語に対応しています。
ここに【例：スペイン語／タイ語】を追加してください。

- 言語切替バーにボタンを追加
- 各テキストに同じ構成で <span class="l-xx"> を追加し、body[data-lang="xx"] のCSSも追加
- 翻訳は機械的な直訳ではなく、その言語のゲストに自然で分かりやすい表現に

修正後のHTMLを丸ごと出力してください。
```

## ⑤ ハウスルールの「角が立たない」言い回しを考えてもらう

```
民泊のハウスガイドに載せる「ハウスルール」を、ゲストが気持ちよく守れる、
やわらかいけれど伝わる言い回しで書いてください。

守ってほしいこと：【例：禁煙、22時以降は静かに、ゴミは分別、定員厳守、又貸し禁止…】
トーン：【例：歓迎ムードを保ちつつ、大事なルールは明確に】

日本語・英語・中国語・韓国語・フランス語で、それぞれ箇条書きにしてください。
特に大事なもの（火気・騒音・定員）は強調表現にしてください。
```

## ⑥ 緊急・防災情報を正しく作ってもらう

```
私の物件【住所：例 長野県○○高原】のハウスガイドに載せる「緊急時セクション」を作ってください。

- 地震・火事・体調不良それぞれの初動を、短い手順で
- 最寄りの病院・消防/警察・指定避難場所の「調べ方」も教えてください
  （※実際の住所・電話番号は私が確認して入れます。架空の情報は入れないでください）
- 5言語で、落ち着いて読める表現に

まず日本語版を作り、私が病院名などを確定したら他言語に展開してください。
```

> ⚠️ 緊急・防災情報は**必ず自分で実在の正確な情報を確認**して入れてください。AIに住所や電話番号を創作させないこと。

## ⑦ 周辺・買い物情報を埋めてもらう

```
【物件の場所：例 長野県○○高原】の近くにある、滞在中のゲストが助かる場所を挙げてください。
- コンビニ・スーパー（営業時間・距離つき）
- おすすめの飲食店3つ（ジャンル・距離つき）
- 雨の日でも楽しめるスポット
そのうち主要なものを、ハウスガイドの「周辺情報」欄に載せる短い表現にまとめ、
日本語・英語で出してください。
```

---

## ⑧ Wi-Fi・暗証番号の「安全な載せ方」を相談する

```
ハウスガイドに Wi-Fiパスワードや鍵の暗証番号を載せるか迷っています。
このガイドは【例：QRコードで滞在中のゲストにだけ共有／URLは検索に出さない】予定です。
情報漏れのリスクと、安全な運用方法（載せる/載せない、別途メッセージで送る等）を、
初心者にも分かるように整理して、おすすめの方法を教えてください。
```

## ⑨ QRコード化・無料公開を手伝ってもらう

```
完成したハウスガイドHTMLを、ゲストがスマホで開けるように公開したいです。
- いちばん簡単な無料の公開方法（GitHub不要の方法も）を手順で
- 公開URLをQRコードにする無料の方法
- 冷蔵庫や玄関への貼り方・予約メッセージへの載せ方のコツ
を、初心者向けに教えてください。
```
