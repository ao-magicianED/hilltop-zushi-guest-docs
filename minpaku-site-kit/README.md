# 🏡 民泊サイト制作キット（Minpaku Site Kit）

**プログラミング不要**で、自分の民泊・宿のおしゃれな公式サイトが作れるテンプレートです。
AI（ChatGPT / Claude など）に手伝ってもらえば、**写真と文章を用意するだけ**で完成します。

> このキットは、実際に運営している民泊サイト「Hilltop Zushi」で使ったノウハウを、
> だれでも再利用できる形にまとめたものです。セミナー配布用に公開しています。

---

## ✨ このサイトの特徴

| 特徴 | かんたんに言うと |
|---|---|
| 📄 **ファイル1つで完成** | `.html` を1個ダブルクリックすれば開く。難しい環境構築はゼロ |
| 🌍 **多言語に対応** | 日本語・英語・中国語など、ボタン1つで切り替え（外国人ゲスト対応） |
| 📱 **スマホ対応** | パソコンでもスマホでも、自動できれいに表示 |
| 🖼 **写真ギャラリー** | カテゴリ絞り込み＋クリックで拡大表示（ライトボックス） |
| 💴 **完全無料で公開できる** | サーバー代0円でインターネットに公開する方法を後述 |
| 🔌 **特別なソフト不要** | 「フレームワーク」や「ビルド」など難しい仕組みを使っていません |

※ **フレームワーク／ビルド** … プロが使う組み立てツールのこと。これがあると初心者には難しくなりますが、このキットは**あえて使っていない**ので、`.html`ファイル単体で動きます。

---

## 📦 中身（ファイルの説明）

```
minpaku-site-kit/
├─ index.html         ← 最初に開く入口ページ（見本・テンプレへの入口）
├─ README.md          ← いま読んでいるこの説明書
├─ PROMPTS.md         ← AIにサイトを作らせる「コピペ用の指示文」集 ★おすすめ
├─ template.html      ← 白紙のテンプレート（穴埋めで作る用）
└─ example-tokyo.html ← 完成見本（東京・夜景の民泊。雰囲気の参考に）
```

## 🚀 まずは見てみる（30秒）

1. `example-tokyo.html` を**ダブルクリック**する
2. ブラウザ（Chrome等）で完成見本が開きます
3. 右上の「JA / EN」で**言語が切り替わる**のを試す
4. 写真をクリックすると**拡大表示**されるのを試す

> 見本の写真は「仮の画像（グレーの枠）」です。実際は自分の写真に差し替えます。

## 🛠 自分の物件用に作る（3ステップ）

まず `template.html` を**コピー**し、ファイル名を `index.html`（または `my-site.html`）に変えます。そのコピーを開いて、次の3か所を変えるだけです（原本はそのまま残しておくと安心）。

### ステップ1️⃣：文章を変える
- `template.html` を「メモ帳」や「VS Code」などで開く
- 一番下の方にある `i18nData` の中の、**`【　】` で囲まれた部分**を自分の物件の情報に書き換える
- 日本語（`ja`）と英語（`en`）の2か所があります

### ステップ2️⃣：写真を入れる
- サイトと同じ場所に `images` フォルダを作り、写真を入れる
- HTMLの中の `<img data-ph="📷|メイン写真 1" alt="">` を
  `<img src="images/写真の名前.jpg" alt="">` に書き換える
- （写真がまだなくても、仮画像のままサイトは完成します）

### ステップ3️⃣：色を変える
- ファイル上の方にある `:root` の**色コード（`#A8895C` など）**を変える
- 例：海の宿なら青系、和の宿なら茶系、など物件の雰囲気に合わせて

> 💡 **むずかしければ、ステップ1〜3を全部AIにやってもらえます。** → 次へ

## ✅ 公開前チェックリスト（だいじ）

公開ボタンを押す前に、ここだけは確認しましょう。**差し替え忘れ**が起きやすいポイントです。

- [ ] 物件名・住所は自分のものになっているか
- [ ] **予約ボタンのリンク先**（Airbnb / Booking.com）を自分の予約ページに変えたか
- [ ] **お問い合わせのメールアドレス**を自分のものに変えたか
- [ ] **Googleマップ**が自分の物件の場所になっているか（テンプレの初期値は仮の場所です）
- [ ] サンプルの文章（「これは見本です」など）が残っていないか
- [ ] スマホで開いて、文字切れ・崩れがないか
- [ ] 写真が重すぎないか（横1600px程度に縮小）
- [ ] 誤字脱字（とくに英語・外国語）

## 🌐 インターネットに無料で公開する

作った `.html` を、世界中から見られるようにする方法です。**いちばん簡単な順**に紹介します。

### 方法A：ドラッグ&ドロップで公開（初心者おすすめ・GitHub不要）
1. [Netlify Drop](https://app.netlify.com/drop) または [Cloudflare Pages](https://pages.cloudflare.com/) を開く
2. サイトのフォルダを**画面にドラッグ&ドロップ**する
3. 数十秒で `https://〇〇〇.netlify.app` のような**公開URL**が発行される

> 公開したサイトのファイル名を `index.html` にしておくと、URLがきれいになります。

### 方法B：GitHub Pages（このキットをGitHubで受け取った人向け）
1. GitHubで自分のアカウントにこのリポジトリを置く
2. リポジトリの **Settings → Pages** を開く
3. 公開元のブランチを選んで保存 → 数分で公開URLが出ます

## 📸 写真のコツ（サイトの印象は写真で9割決まります）

- **横向き（横長）** の写真を使う（縦写真は切れて見えます）
- 明るい時間帯・**自然光**で撮る
- 1枚目（ヒーロー）は**いちばん自慢の1枚**を
- スマホ撮影でも十分。撮ったら明るさを少し上げるだけで見違えます
- 容量が大きすぎると重いので、**横1600px程度**に縮小すると◎

## ❓ よくある質問

**Q. プログラミングの知識は必要？**
A. 不要です。文字を書き換えるだけ。むずかしければAIに頼めます。

**Q. お金はかかる？**
A. キットも公開も無料です。独自ドメイン（`〇〇.com`）が欲しい場合だけ年1,000〜2,000円ほど。

**Q. 予約システムはついてる？**
A. 予約は Airbnb / Booking.com 等の**自分の予約ページにリンク**する形です（このサイトは「お店の顔」になる公式ページ）。

**Q. スマホだけで作れる？**
A. 文章の修正くらいなら可能ですが、**パソコン推奨**です（ファイルの編集がしやすいため）。
