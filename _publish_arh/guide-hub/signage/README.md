# Hilltop Zushi 壁掲示物セット（A4縦・4言語対応）

民泊施設に貼る印刷用掲示物です。すべて **A4縦・日本語/English/中文/한국어** の4言語対応。

---

## 📦 同梱ファイル

| ファイル | 用途 | 推奨貼り場所 |
|---|---|---|
| `no-smoking-A4.html` | 全館禁煙の標識 | 玄関ドア内側／リビング入口／各寝室入口 |
| `bbq-rules-A4.html` | BBQ利用ルール（10:00〜20:00、騒音配慮） | 庭BBQエリアの目に入る場所／勝手口内側 |
| `insects-notice-A4.html` | 虫の案内＋殺虫剤利用案内 | キッチン壁／サンルーム／殺虫剤の置き場所近く |

---

## 🖨️ 印刷手順（Windows + Chrome 想定）

1. 該当HTMLファイルをダブルクリック → Chromeで開く
2. **Ctrl + P** で印刷ダイアログを開く
3. 以下を設定:
   - **送信先**: お使いのプリンタ
   - **用紙サイズ**: A4
   - **レイアウト**: 縦
   - **余白**: なし（または最小）
   - **倍率**: 既定 / 100%
   - **背景のグラフィック**: ✅ ONにする（重要・色背景が出ない場合の対策）
4. **印刷** ボタン

> 💡 **背景色やフレームが出ない場合**:
> 「詳細設定」→「オプション」→「**背景のグラフィック**」にチェック。

## 📐 用紙・額装のおすすめ

### コスパ重視（家庭用プリンタ）
- 用紙: A4 厚手マットコピー紙（コクヨ KB-39N など）
- 額: ニトリ・無印のA4額（白 or 木目）

### 見映え重視
- 用紙: キヤノン両面マット紙 / EPSON フォトマット紙
- 額: アクリル額・パネル加工（300円ショップでも可）
- ラミネート加工すれば水・汚れに強い（庭のBBQ用は特に推奨）

### 業者印刷（大量・キレイ）
- ラクスル・プリントパックで「**チラシ印刷 A4 マットコート135kg**」
- 1枚あたり10〜30円程度（10枚〜）

## 🌐 4言語の根拠（チェック観点）

| 言語 | フォント | 確認ポイント |

## 🏷 ブランド・タイポグラフィ運用

### ヘッダー表記方針
**現在の掲示物は「テキストのみ」のシンプルなヘッダー** を採用しています。
ロゴ画像（アイコン＋フォント）は別の場面（公式サイト・名刺・SNS等）で使用想定。

掲示物ではテキスト表記に統一する理由:
- A4印刷時の鮮明さを最優先（小さなアイコン部の潰れを回避）
- 4言語の情報量が多いため、視覚ノイズを減らす
- モノクロ印刷でもブランド名が確実に読める
- アイコン側に山・葉っぱのモチーフがあるため、文字側はシンプルに（バランス重視）

### 文字仕様
- **フォント**: Cormorant Garamond（700 weight） / Noto Serif JP fallback
- **カラー**: `--primary` (#1F3A34 Deep Forest Green)
- **letter-spacing**: 0.16em（上品な字間）

### 各掲示物の文字サイズ
| 掲示物 | font-size | 理由 |
|---|---|---|
| 禁煙標識 | 1.95rem | 中央レイアウト、最も主張させる |
| BBQルール | 1.6rem | 情報密度が高いので控えめに |
| 虫の案内 | 1.75rem | 上品さを保ちつつ視認性確保 |

### ロゴ画像アセット（参考用・現状未使用）
オリジナル: `signage/assets/`
- `logo-horizontal.png` — アイコン＋フォント横並び
- `logo-icon.png` — アイコンのみ

オリジナルのロゴシステムデザイン → `神武寺民泊PJ/アイコン・ろご/ロゴシステムデザイン.png`
