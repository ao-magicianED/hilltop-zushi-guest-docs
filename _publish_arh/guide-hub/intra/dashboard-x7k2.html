🚨 事象別 緊急対応SOP Emergency SOP CRITICAL

住宅宿泊事業者は苦情・トラブルに30分以内に現地対応する義務あり。各SOPは「誰が・何分以内に・何を」まで明文化済。

🔥 1. 火災・煙感知 担当: 第一発見者→あお
⏱ SLA: 即119・5分以内に運営者通知・30分以内現地
  1. 即119通報（住所「神奈川県逗子市池子2-10-9 Hilltop Zushi」）
  2. ゲスト全員を屋外退避（玄関→下り坂方向）
  3. 初期消火可能なら消火器、不能なら退避優先
  4. あお → 空ちゃん → 黒須氏（インビックス）の順で連絡
  5. 藍コーポレーション 046-874-7178 へ報告
  6. 火災後: 警察・保険会社へ通報、ゲスト写真・状況をログ化
📞 ゲストへの初動: 「Hilltop Zushi 運営です。すぐ建物から出てください。玄関から坂を下りる方向です。今119通報しました。」
🌏 2. 地震（震度4以上） 担当: ゲスト自助→あお
⏱ SLA: 揺れ収まり次第即連絡・1時間以内現地確認
  1. ゲストは「机の下・建物中央へ」（ハウスガイドv3にも記載）
  2. 収まったらガス元栓・電気ブレーカー確認 → 止め方マップ
  3. 給湯器（プロパン）漏れ・異臭確認
  4. 建物に亀裂・傾きがあれば即避難（旧耐震）
  5. 避難所: 逗子市立池子小学校 / 久木中学校
  6. 翌日: 専門業者による構造点検手配
💨 3. ガス漏れ・警報器作動 担当: 第一発見者→サガミ
⏱ SLA: 即対応・5分以内サガミ通報
  1. 火気厳禁（電灯スイッチも触らない）
  2. 窓を開けて換気
  3. 元栓（屋外プロパンボンベ）を閉める → 止め方マップ
  4. サガミ 046-833-3131
  5. ゲスト全員を屋外退避
📞 サガミへの台本: 「Hilltop Zushi 池子2-10-9 です。ガス漏れの疑いで全員退避中。元栓は閉めました。至急お願いします。」
🚑 4. ゲスト負傷・急病 担当: 同行者→119→あお
⏱ SLA: 重症は即119・軽症は1時間以内現地
  1. 意識・呼吸を確認 → 重症なら即119
  2. 軽傷: 救急箱 ⚠ 場所要明記
  3. 夜間・休日: 横須賀共済病院 046-822-3433（二次救急24h）
  4. Airbnb/Bookingメッセージで状況記録（保険請求用）
  5. インシデントログに記入
⚠ 5. 崖からの転落（パノラマテラス側） 担当: 第一発見者
⏱ SLA: 即119・動かさない
  1. 119通報 → 救急隊到着まで動かさない
  2. 意識確認 → 出血があれば清潔な布で圧迫止血
  3. 「掲示物 #2 立入禁止警告」が機能していたか写真記録（証跡）
  4. 保険会社（火災保険・施設賠償特約）へ即連絡
  5. インシデントログ 詳細記入・本件は保健所報告も検討
⚡ 6. 停電 担当: あお
⏱ SLA: 1時間以内に状況把握・ゲスト連絡
  1. 近隣も停電か確認 → 地域停電なら東京電力 0120-995-007
  2. 個別ブレーカー: 場所 ⚠ 要明記 → 止め方マップ
  3. 給湯器再点火確認
  4. 1時間以上復旧しないならゲストへ謝罪 + 状況説明
  5. 2時間以上→宿泊代金部分返金検討（要協議）
💧 7. 断水・漏水 担当: あお→藍コーポ
  1. 漏水: 元栓（玄関外メーターボックス）を閉める → 止め方マップ
  2. 神奈川県営水道 0467-22-6200
