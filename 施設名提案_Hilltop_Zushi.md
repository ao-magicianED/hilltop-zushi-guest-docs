# 施設名提案 / Property Name Proposals

**物件所在地:** 逗子市池子2-10-9（JR東逗子駅 徒歩約90秒）
**物件タイプ:** 一軒家（最大12名）
**コンセプト:** 森の隠れ家・自然に囲まれた一軒家リトリート

---

## 選定基準

| 基準 | 説明 |
|------|------|
| 英語での響き | 英語ネイティブが発音しやすく、覚えやすいか |
| 検索性（SEO） | Airbnb/Booking.com内検索で埋もれないユニークさ |
| ブランディング | コンセプト（森・隠れ家・自然）との一致度 |
| 文化的魅力 | 日本文化への憧れを刺激するか |
| OTA表示映え | リスティングタイトルに収まるか（全角30文字以内推奨） |

## 提案一覧

### 案1: 花隠れ（Hanakakure）★推奨

| 項目 | 内容 |
|------|------|
| **読み** | はなかくれ / Hanakakure |
| **意味** | "Hidden among flowers" — 花に隠れた場所 |
| **コンセプト** | 四季折々の花と緑に囲まれ、都市の喧騒から「隠れる」ように佇む一軒家。日本庭園や里山の情緒を連想させる。 |
| **英語での印象** | "Hana"（花＝日本語として広く認知）+"kakure"（隠れ）。『NARUTO』の木ノ葉隠れ（Konohagakure）との語感的な親和性があり、日本文化ファンに響く。 |
| **Airbnb表示例** | `Hanakakure — Hidden Forest Retreat near Kamakura` |
| **Booking.com表示例** | `花隠れ Hanakakure - Zushi Forest House` |
| **強み** | 唯一無二のネーミング。検索競合なし。覚えやすい4音節。 |
| **弱み** | 初見では意味が伝わりにくいため、サブタイトルでの補足が必要。 |

### 案2: 森の庵（Mori no Iori）

| 項目 | 内容 |
|------|------|
| **読み** | もりのいおり / Mori no Iori |
| **意味** | "Forest hermitage" — 森のなかの庵（いおり） |
| **コンセプト** | 鎌倉時代の草庵・茶室文化を連想させる、質素ながら趣のある隠れ処。禅的な世界観。 |
| **英語での印象** | "Mori"（森）は直感的。"Iori"は女性名としても通用し柔らかい響き。 |
| **Airbnb表示例** | `Mori no Iori — Forest Hermitage, Zushi` |
| **Booking.com表示例** | `森の庵 Mori no Iori - Traditional House in Zushi` |
| **強み** | 「森」のイメージが物件の立地と直結。和の趣が強い。 |
| **弱み** | "Iori"の発音が英語話者にやや難（イオリ vs アイオリ）。「庵」の本来の意味（質素な小屋）が一軒家12名規模と若干ギャップ。 |

### 案3: 木漏れ日（Komorebi）

| 項目 | 内容 |
|------|------|
| **読み** | こもれび / Komorebi |
| **意味** | "Sunlight filtering through leaves" — 木々の間から差し込む光 |
| **コンセプト** | 森に囲まれた物件に朝の光が差し込む情景そのもの。海外で「翻訳できない美しい日本語」として広く知られる単語。 |
| **英語での印象** | BBC、NYT等で「世界で最も美しい日本語」として紹介実績あり。認知度が高く、憧れを刺激する。 |
| **Airbnb表示例** | `Komorebi — Forest Light House near Kamakura` |
| **Booking.com表示例** | `木漏れ日 Komorebi - Sunlit Forest Home, Zushi` |
| **強み** | 海外認知度No.1の日本語ワード。検索でも「komorebi」で辿り着ける可能性。BBQと森の組み合わせにぴったり。 |
| **弱み** | 同名の宿泊施設が既に複数存在（京都・箱根等）。差別化にサブタイトルの工夫が必要。 |

### 案4: 逗子庵（Zushi-an）

| 項目 | 内容 |
|------|------|
| **読み** | ずしあん / Zushi-an |
| **意味** | "Zushi retreat" — 逗子の庵 |
| **コンセプト** | 地名を冠することでロケーションを即座に伝える実用的なネーミング。「庵」が和のテイストを加える。 |
| **英語での印象** | 地名＋日本語の組み合わせは分かりやすい。"an"は短く発音しやすい。 |
| **Airbnb表示例** | `Zushi-an — Traditional House with BBQ Garden` |
| **Booking.com表示例** | `逗子庵 Zushi-an - Japanese House near Kamakura` |
| **強み** | SEO最強。「Zushi」で検索するゲストにダイレクトに刺さる。覚えやすい。鎌倉エリアへのアクセスも暗示。 |
| **弱み** | ポエティックさに欠ける。「逗子」の知名度は鎌倉と比べ低い。 |

### 案5: 翠隠（Suiin）

| 項目 | 内容 |
|------|------|
| **読み** | すいいん / Suiin |
| **意味** | "Hidden in green" — 翠（みどり）に隠れた場所 |
| **コンセプト** | 「翠」（翡翠の翠、深い緑）＋「隠」（隠れる）。花隠れの姉妹候補として、より静謐で高級感のある響き。 |
| **英語での印象** | 短く、ミステリアスな響き。日本の旅館名のような格調。 |
| **Airbnb表示例** | `Suiin — Hidden Green Retreat, Zushi` |
| **Booking.com表示例** | `翠隠 Suiin - Forest Villa near Kamakura & Hayama` |
| **強み** | 完全にユニークで検索競合ゼロ。漢字2文字の簡潔さ。高級路線にも対応可能。 |
| **弱み** | 読めない漢字（翠）。意味の伝達にサブタイトル必須。 |

## 推奨

**「花隠れ（Hanakakure）」を第一推奨**とします。

- ユニーク性・ブランド力・文化的魅力のバランスが最も優れている
- OTAリスティングでの表示例: **Hanakakure — Hidden Forest Retreat near Kamakura**
- サブタイトルで「森の隠れ家」「BBQ付き一軒家」等を補足可能
- ドメインやSNSアカウントの取得もしやすい

**次点:** 木漏れ日（Komorebi）— 海外認知度の高さは突出しているが、既存競合との差別化が課題。
